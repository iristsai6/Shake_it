# 🍸 走跳酒 Bar 
這是一款專為微醺愛好者打造的單頁式 Web 應用程式 (SPA)，結合了調酒百科、互動遊戲、飲酒安全追蹤以及高雄在地酒吧指南。
## 🌟 核心功能
### 1.Home-隨機配方生成
 * **酒譜探索**：提供多款經典調酒配方，包含基酒、風味、酒感強度（ABV）與調酒師建議。
 * **搖晃感應**：支援手機物理搖晃（Shake to Generate）觸發生成，模擬真實調酒體驗（iOS 使用者需點擊按鈕授權）。
### 2. Home-派對大冒險
 * **互動指令**：內建多組適合酒吧現場的抽籤指令。
 * **全自訂模式**：使用者可根據需求新增、刪除或清空自訂的大冒險指令。
### 3.Guide-調酒指南
 * **基酒分類**：系統化整理琴酒、威士忌、蘭姆酒、龍舌蘭、伏特加、白蘭地六大基酒。
 * **AI 助手介面**：提供與 AI 調酒師對話的輸入介面，讓使用者透過自然語言描述（如：喜歡果香、怕苦等）尋找推薦。
### 4. Tracker-飲酒計數與生存守則
 * **實時計數**：追蹤當晚飲酒杯數，並根據杯數切換視覺狀態（綠色安全 / 紅色警示）。
 * **補水機制**：當杯數超過 3 杯時，系統會自動觸發「補水提醒」動畫，提倡安全飲酒。
 * **酒客守則**：提供多項安全清單（如：不空腹、不酒駕等）供使用者勾選。
### 5. Map-酒吧探索
 * **酒吧百科**：分類介紹五大類型酒吧（Cocktail, Whisky, Wine, Craft Beer, Speakeasy）。
 * **高雄地圖推薦**：預載高雄精選酒吧清單，點擊後自動跳轉至 Google Maps 搜尋。
## 🛠 技術架構與開發者說明
### 技術組成
 * **前端**：原生 HTML5, CSS3 (CSS Variables, Flexbox), JavaScript (ES6+)。
 * **感應器**：整合 DeviceMotionEvent 偵測行動裝置位移速度。
 * **地圖整合**：使用動態 URL Scheme [https://www.google.com/maps/search/](https://www.google.com/maps/search/) 實踐地圖導引。
### ⚠️ API 串接狀況
目前程式碼中關於 **Claude AI 助手** 的對話功能僅完成前端介面與邏輯開發：
 * **現狀**：fetch 請求中的 apiKey 目前為空字串，且端點直接發送至 Anthropic 伺服器。
 * **建議**：若要啟用此功能，開發者需自行填入有效的 API Key，或將請求改由後端 Proxy 轉發以確保密鑰安全。
## 📱 使用說明
 * **本地執行**：下載單一 HTML 檔案後，直接使用瀏覽器開啟即可。
 * **iOS 搖晃感應**：初次使用請在首頁點擊「點我啟用搖晃」，並在彈出的系統視窗中點擊「允許」。
